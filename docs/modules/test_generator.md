# 测试生成模块

测试生成模块从 DBC 文件自动生成测试用例，支持多种测试方法。

## 功能

- **自动用例生成**：从信号属性自动生成边界值、正常范围、超时等测试用例
- **用例列表**：按分类/优先级过滤
- **覆盖率仪表盘**：信号覆盖率统计
- **导出**：Excel、JSON

## 测试方法

| 方法 | 说明 |
|------|------|
| 边界值测试 | 最小/最大物理值 |
| 正常范围测试 | 中间值验证 |
| 超时测试 | 报文停发后的超时处理 |
| 异常值注入 | 超出物理范围的原始值 |

## 操作流程

### 生成用例

1. 加载 DBC 文件（拖放或导入）
2. 点击 **生成测试用例**
3. 选择测试方法
4. 自动生成并显示在列表中

### 查看覆盖率

- 覆盖率面板显示：总信号数、已覆盖数、覆盖率百分比
- 未覆盖信号高亮提示

### 导出

- **Excel**：包含用例 ID、名称、分类、步骤、预期结果等 13 列
- **JSON**：包含用例数据 + 覆盖率统计
