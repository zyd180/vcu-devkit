# 需求追溯模块

需求追溯模块用于建立需求与工程工件之间的追溯关系。

## 功能

- **追溯矩阵表格**：需求 × 链接类型的交叉表，红绿色标记覆盖状态
- **需求详情面板**：查看/编辑需求属性
- **链接管理**：添加/删除 SWC、信号、DTC、测试用例链接
- **统计仪表盘**：追溯覆盖率、未追溯需求列表
- **Excel 导出**：完整的追溯矩阵报告

## 操作流程

### 导入需求

通过数据库管理需求数据（支持从飞书多维表格导入）。

### 建立链接

1. 选中需求
2. 点击 **添加链接**
3. 选择链接类型（SWC / 信号 / DTC / 测试用例）
4. 选择目标工件

### 查看追溯状态

- 绿色：已追溯（有链接）
- 红色：未追溯（无链接）
- 覆盖率面板显示百分比

### 导出

导出 Excel 追溯矩阵，包含需求 ID、描述、各类型链接状态。
