# CAN 开发模块

CAN 开发模块用于浏览、编辑 DBC 文件，并生成 C/CAPL 代码。

## 界面布局

```
┌──────────┬──────────────────────────────────┐
│ 报文树    │  信号列表（TableEditor）          │
│ (TreeView)│  ┌─────────────────────────────┐ │
│          │  │ 信号属性面板（PropertyPanel）  │ │
│ VCU      │  └─────────────────────────────┘ │
│  ├ Status │                                  │
│  ├ Torque │  工具栏：打开/保存/Diff/校验/生成  │
│  └ HV     │                                  │
└──────────┴──────────────────────────────────┘
```

## 操作流程

### 导入 DBC

1. 拖放 `.dbc` 文件到窗口
2. 或点击工具栏 **打开** 按钮
3. 文件解析后，左侧树按发送方分组显示报文和信号

### 编辑信号

1. 在左侧树或信号列表中选中信号
2. 右侧属性面板显示：名称、起始位、位长、字节序、值类型、Factor、Offset、最小值、最大值、单位、注释、接收方
3. 直接修改字段值
4. 每次修改自动记录到撤销历史

### 添加/删除信号

- 工具栏 **+** 按钮：在当前报文中添加新信号
- 工具栏 **-** 按钮：删除选中信号

### 校验

按 `F5` 运行以下规则：

| 规则 ID | 严重级别 | 说明 |
|---------|---------|------|
| DBC_OVERLAP | 错误 | 信号位域重叠 |
| DBC_DLC_SHORT | 错误 | DLC 不足以容纳信号 |
| DBC_DUP_ID | 错误 | 重复的报文 ID |
| DBC_MIXED_ENDIAN | 警告 | 报文内混合字节序 |
| DBC_RANGE_BELOW | 警告 | 信号最小值不可达 |
| DBC_RANGE_ABOVE | 警告 | 信号最大值不可达 |
| DBC_NAME_INVALID | 警告 | 名称含非法字符 |
| DBC_UNSIGNED_NEG_OFFSET | 提示 | 无符号信号有负偏移 |

### Diff 对比

点击工具栏 **Diff** 按钮，选择另一个 DBC 文件进行版本对比。

### 保存

- `Ctrl+S`：保存为 DBC 格式（使用 cantools 库）
- 支持保留：信号转换、值描述、多路复用、注释、扩展帧

### 生成代码

按 `F6` 生成 C 头文件：

- `can_pack.h` — Pack/Unpack 函数声明
- `can_signals.h` — 信号结构体定义
- `can_messages.h` — 报文 ID 和 DLC 定义

支持大端/小端、有符号/无符号、缩放因子。
